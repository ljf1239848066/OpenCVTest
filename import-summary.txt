Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* aidl/ => opencvlib/src/main/java/
* aidl/org/opencv/engine/OpenCVEngineInterface.aidl => opencvlib/src/main/aidl/org/opencv/engine/OpenCVEngineInterface.aidl
* src/ => opencvlib/src/main/java/
* src/.DS_Store => opencvlib/src/main/resources/.DS_Store
* src/org/.DS_Store => opencvlib/src/main/resources/org/.DS_Store
* src/org/opencv/.DS_Store => opencvlib/src/main/resources/org/opencv/.DS_Store
